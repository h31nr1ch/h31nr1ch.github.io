Graduated in Computer Science (UDESC, 2017), MSc in Applied Computing (UDESC, 2019), and currently a Ph.D. candidate in Informatics (UFPR, 2023). My research interest is mainly focused on topics of security. In network security, I work with measurements and evaluation to define the behavior of attacks such as DRDoS. Implementing systems like honeypots to collect attacker's interactions and package evaluation. In host security, I work with anomaly detection through the use of machine learning strategies. Currently, I am working on a strategy for intrusion detection in the WebAssembly sandbox. These solution uses data categorization to improve the learning phase of the machine learning models.
